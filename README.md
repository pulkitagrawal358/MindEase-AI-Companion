Overview
This project was developed during my AICTE - Edunet Foundation Internship (January 2026 Batch). It is a Python-based chatbot designed to support student well-being by detecting emotional states through sentiment analysis and providing empathetic responses.

Features
- Sentiment Analysis: Uses NLP to categorize user input into Positive, Negative, or Neutral moods.
- Real-time Interaction: Provides instant motivational feedback and relaxation tips.
- Dataset Integration: Built and tested using a comprehensive Mental Health Sentiment dataset.

Project Structure
- mental_health_bot.ipynb: Core Python code and development steps.
- Combined Data.csv: The dataset used for testing and validation.
- Project_Presentation.pdf: Detailed project walkthrough and results.
- Code_Export.pdf: Final code documentation.

How to Run
1. Clone this repository.
2. Install requirements: pip install pandas textblob.
3. Run the cells in mental_health_bot.ipynb.
